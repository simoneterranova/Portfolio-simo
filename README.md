```bash

```

2. Navigate to the project directory:
```bash
cd [repository-name]
```

3. Install dependencies:
```bash

# or
bun install
```

4. Start the development server:
```bash

# or

```

5. Open [http://localhost:8080](http://localhost:8080) in your browser
